# Liuli Society — Classic Theme Website

A GitHub-ready React + Vite version inspired by a classic nonprofit / institutional website style.

## Design direction
- classic, warm, and refined
- editorial serif headings
- more traditional layout rhythm
- different footer design
- realistic-style image banners included in `src/assets`

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Hosting
The Vite config uses `base: './'` so it is easier to host as a static website.
